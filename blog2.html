1. What Are Tags?

Tags in HTML are like labels. They tell the browser how to display content. For example, <p> is used for paragraphs, and <h1> is for main headings.

2. Common Tags I Learned

  • <h1> to <h6>: Headings
  • <p>: Paragraphs
  • <a>: Links
  • <img>: Images
  • <ul>, <li>: Lists

3. Nesting Tags

You can put tags inside other tags. For example, a link can go inside a paragraph. Nesting helps you build complex layouts using simple parts.
