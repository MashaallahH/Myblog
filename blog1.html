How I Built My First Website

Category: HTML

Posted on April 15, 2025

I always wanted to create a website, but I had no idea where to start. So I decided to begin with the basics: HTML and CSS.

1. Starting with HTML

HTML is the foundation of every website. I learned how to use tags like <h1>, <p>, and <a> to add content to my page. I also used <img> to insert images.

2. Adding CSS for Style

Once I had the structure, I used CSS to make it look better. I changed fonts, colors, and added spacing. I also made a simple layout using margins and padding.

3. Seeing It All Come Together

When I opened the file in my browser and saw my website come to life, it felt amazing! It was simple, but it was mine. And I built it from scratch.

4. What's Next?

Now that I understand the basics, I'm planning to learn JavaScript and make my website interactive. The journey has just begun!
